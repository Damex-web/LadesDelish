# 🍽️ Lades_Delish Website

A modern and responsive food business website built with **pure HTML, CSS, and JavaScript** for **Lades_Delish** — a catering and food delivery brand offering event food coverage, mini party catering, and home delivery services.

---

# 🌟 Features

- ✅ Fully Responsive Design
- ✅ Modern Food Business UI
- ✅ Hero Landing Section
- ✅ Services Showcase
- ✅ Food Gallery
- ✅ About Section
- ✅ Testimonials
- ✅ Contact Form
- ✅ SEO Optimized
- ✅ Mobile Friendly
- ✅ Smooth Scrolling Navigation

---

# 🍔 Services Included

- Event Catering
- Mini Party Food Coverage
- Home Food Delivery
- Local & Continental Dishes
- Cocktails & Mocktails
- Event Management

## 1. Clone the repository

```bash

```

## 2. Open the project folder

```bash
cd lades_delish
```

## 3. Run the website

Open `index.html` in your browser.
